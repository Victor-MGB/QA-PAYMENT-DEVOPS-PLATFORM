# QA DevOps Platform

A production-grade QA automation platform deployed on AWS.

## Stack
- **API Testing**: Jest + Supertest
- **UI Testing**: Cypress
- **Performance**: k6
- **Infrastructure**: Terraform + Ansible + AWS
- **CI/CD**: GitHub Actions
- **Monitoring**: Bash watchdog + Cron

## Getting Started
```bash
npm install
npm run test:api
npm run test:ui
```
