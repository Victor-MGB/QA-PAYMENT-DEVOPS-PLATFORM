# QA payment DevOps Platform

A production-grade QA payment automation platform deployed on AWS.

## Stack
- API Testing: Jest + Supertest
- UI Testing: Cypress
- Performance: k6
- Infrastructure: Terraform + Ansible + AWS
- CI/CD: GitHub Actions
- Monitoring: Bash watchdog + Cron

## Getting Started
npm install
npm run test:api
npm run test:ui
